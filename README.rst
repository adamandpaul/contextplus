Development Tools
=================

Development tools can be built by running::

   bin/bootstrap
   bin/build

Tools that get built:

- ``bin/test`` - run the test suite
- ``bin/code-analysis`` - run the linter

Documentation that gets built can be found in the directory ``buildout/parts/docs``
